Basic information on how to use this tool:
- executable: `pling`
- help: `-h`, `--help`
- version: `--version`
- description: 
>Pling is a software workflow for plasmid analysis using rearrangement distances, specifically the Double Cut and Join Indel (DCJ-Indel) distance. By intelligently combining containment distance (shared content as fraction of the smaller) and DCJ-indel distance (“how far apart evolutionarily” in a structural sense), and by preventing shared mobile elements from clouding the issue, it infers clusters of related plasmids.

Additional information:

Please note that on top of `pling cluster`, there are two additional main subcommands for pling. Namely, `pling submatrix` and `pling add`. Unfortunately, although the documentation mentions them under advanced usage, there is little to no documentation about them presently available. We strongly suggest that the user run the following commands to learn about their respective use cases based on their needs:

- `pling add --help`: "This command is for adding novel plasmids to an existing pling clustering, or merging multiple prior clustering together."
    - `pling add [OPTIONS] GENOMES_LIST OUTPUT_DIR PREVIOUS_PLING...`
    - Add new genomes to an existing pling clustering, and/or merge multiple previous clusterings into a single clustering.
    - First input is a path to list of fasta file paths. This list must contain ALL genomes to be clustered, i.e. both genomes to add to previous clustering and genomes from previous clustering.
    - Second input is a path to an output directory.
    - Third input is a path to previous pling output directory (multiple are permitted).
- `pling submatrix --help`: "Provides some potentially useful outputs for downstream analysis, based on pling clustering results and DCJ-Indel distances."
    - `pling submatrix [OPTIONS] OUTPUT_DIR`
    - Output DCJ-Indel distances as matrices for each subcommunity, and calculate neighbour-joining trees.

Frolova, D., Lima, L., Roberts, L. W., Bohnenkämper, L., Wittler, R., Stoye, J., & Iqbal, Z. (2024). Applying rearrangement distances to enable plasmid epidemiology with pling. Microbial Genomics, 10(10). https://doi.org/10.1099/mgen.0.001300

Full documentation: https://pling.readthedocs.io/latest/
