Basic information on how to use this tool:

- executable: `seroba`
- help: `-h`, `--help`
- version: `version`
- description: SeroBA is a k-mer based Pipeline to identify the Serotype from Illumina NGS reads for given references.

Additional information:

- This image contains 71-mer database at `/seroba/database/`
  
Full documentation: https://sanger-pathogens.github.io/seroba/
## Example Usage

```bash
# Create a database for kmc and ariba
seroba createDBs  <database dir> <kmer size>

# Identify serotype
seroba runSerotyping [options]  <databases directory> <read1> <read2> <prefix>

# Summarize the output in one tsv file
seroba summary  <output folder>
```
