# snp-sites container

Main tool: [snp-sites](https://github.com/sanger-pathogens/snp-sites)

Basic information on how to use this tool:

- executable: `snp-sites`
- help: `-h`
- version: `-V`
- description: Finds SNP sites from a multi-FASTA alignment

Full documentation: [https://github.com/sanger-pathogens/snp-sites](https://github.com/sanger-pathogens/snp-sites)

## Example Usage

```bash
snp-sites -o output_snps.fasta input_alignment.fasta
```
