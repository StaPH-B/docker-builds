Basic information on how to use this tool:
- executable: plassembler
- help: --help
- version: --version
- description: |

> plassembler is a program that is designed for automated & fast assembly of plasmids in bacterial genomes that have been hybrid sequenced with long read & paired-end short read sequencing. It was originally designed for Oxford Nanopore Technologies long reads, but it will also work with Pacbio reads. As of v1.3.0, it also works well for long-read only assembled genomes.

Additional information:

The plsdb database that plassembler uses has been pre-downloaded at /plassembler_db using the following command:
```
plassembler download -d /plassembler_db
```
  
Full documentation: https://github.com/gbouras13/plassembler

## Example Usage

```bash
plassembler run \
    -l nanopore.fastq.gz \
    -1 illumina_R1.fastq.gz \
    -2 illumina_R2.fastq.gz \
    -d /plassembler_db \
    -o <output_directory> \
    -t <threads> \
    -c <chromosome size>
```
