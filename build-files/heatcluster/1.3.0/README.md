# HeatCluster container

Main tool: [HeatCluster](https://github.com/erinyoung/heatcluster)
  
Code repository: https://github.com/erinyoung/heatcluster

Basic information on how to use this tool:
- executable: heatcluster
- help: -h
- version: -v
- description: Creates heatmap from SNP matrix generated from SNP dists

Does not currently support molten output, but may in the future.

Full documentation: https://github.com/erinyoung/heatcluster

## Example Usage

```bash
heatcluster -i matrix.csv
```
